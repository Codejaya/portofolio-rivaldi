# 💻 Codejaya Portfolio

Website portofolio modern, responsif, dan SEO-friendly yang di-host di GitHub Pages.

## 📁 Project Structure
- `index.html` - Halaman utama.
- `assets/img/` - Aset gambar profil dan proyek.
- `assets/docs/` - Dokumen pendukung (CV).

## 🚀 Deployment
Akses website di: [https://codejaya.github.io/](https://codejaya.github.io/)

## 🛠️ Built With
- **HTML5** (Semantic)
- **Tailwind CSS** (Styling)
- **JSON-LD** (SEO Data)
